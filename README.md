# Olea – Luxury Café & Restaurant Landing Page

Olea is a modern, premium café and restaurant landing page designed to showcase a luxury dining experience. Inspired by high-end hospitality websites, it combines elegant typography, immersive visuals, smooth animations, and a seamless reservation experience.

##  Features

*  Premium luxury café aesthetic
*  Fully responsive design
*  Built with Next.js and TypeScript
*  Smooth animations with Framer Motion
*  Featured menu with categorized dishes
*  Interactive gallery section
*  Customer testimonials
*  Experience section highlighting events and live music
*  Delhi location and contact information
*  WhatsApp-based reservation system
*  Optimized for deployment on Vercel

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Lucide React

## Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

### Navigate to the project

```bash
cd YOUR_REPOSITORY
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

##  Deployment

This project is optimized for deployment on Vercel.

```bash
npm run build
```

Deploy directly from GitHub through Vercel for automatic deployments on every push.
